Para compilar e executar o projeto, por favor, siga estas etapas:

1.  **Compile o código-fonte:**
    Este comando compila todos os arquivos .cpp da pasta 'src' e cria um arquivo executável chamado 'execGrupo14'.

    ```bash
    g++ src/*.cpp -o execGrupo14 -std=c++11
    ```

2.  **Torne o script de teste executável:**
    Este comando concede ao script 'run_tests.sh' permissão para ser executado.

    ```bash
    chmod +x run_tests.sh
    ```

3.  **Execute o script de teste:**
    Este comando executa o script 'run_tests.sh', faz os testes de todas as instancias com os parametros usados no experimento.

    ```bash
    ./run_tests.sh
    ```

4.  **Execute o programa com uma instância específica:**
    Este comando executa o programa compilado, passando um arquivo de instância específico como argumento.

    ```bash
    ./execGrupo14 instancias_t2/g_60_0.07_0_1_1.txt
    ```

5.  **Interação com o Menu de Heurísticas:**
    Após o início do programa, pressione '3' para entrar no menu de heurísticas. 
    A partir daí, você pode escolher cada uma das heurísticas.
    Os valores de alfa podem ser definidos após a chamada da função.
    Os seguintes valores foram usados nos experimentos:
    Para aquelas que usam um parâmetro alfa, use os valores (0.3, 0.6, 0.9). 
    O número máximo de iterações deve ser definido como 150 e o tamanho do bloco como 30.
    Utilizando os metodos implementados após pressionar 4 e 5, automaticamente será iniciado os testes com os seguintes parametros:
    4- Algoritmos executados 30 vezes, maxIter = 150, Bloco = 30, alfas = {0.3,0.6,0.9}, seed aleatoria.
    5- Algoritmos executados 1 vez, maxIter = 150, Bloco = 30, alfas = {0.3,0.6,0.9}, seed aleatoria.
